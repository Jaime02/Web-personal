Qt

I have been using Qt for a while. It is a extremely powerful framework which provides GUI features and much more. I find it very complete and easy to use. It has three big language APIs: C++, QML and Python.

I love the Python bindings of the C++ API, which are called "PySide". The combination of the capabilities of Qt and the usability of Python is amazing. I have few experience with C++ because I prefer using Python. I would like to learn QML in depth.

I took part in the 2021 Hacktoberferst with the Python Spain discord community and I contributed to the Qt documentation translating some C++ examples to Python. It has been a amazing experience to learn how to carry out all the process. The best part of the experience is knowing a lot of amazing people and receive feedback. I really encourage people to get contributing into open source projects such as Qt, it is very enjoyable.

Due to my contributions, I got elected as one of Qt 2021 champions!
