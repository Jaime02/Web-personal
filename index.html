Proyecto de investigación: "Desarrollo de una aplicación de dibujo técnico"

Este proyecto es mi mayor hito hasta la fecha. Consiste en la creación de un programa informático enfocado al uso educativo del sistema diédrico. El programa ha sido escrito en Python 3. Utiliza PyQt5 para la interfaz y OpenGL para el dibujado 3D. También utiliza Sympy para los cálculos geométricos.
